# CaaT Academy

[![Open CaaT Academy](https://img.shields.io/badge/Open-CaaT%20Academy-0f172a?style=for-the-badge\&logo=githubpages\&logoColor=white)](https://mohammadijoo.github.io/CaaT_Academy/)

CaaT Academy is a browser-native engineering course library for control engineering, robotics, simulation, and mathematical modeling. It provides standalone HTML lessons that can be studied online or downloaded for offline use.

## Website

Visit the live website:

https://mohammadijoo.github.io/CaaT_Academy/

## Course Areas

* Control Engineering
* Robotics
* System Dynamics
* Robot Kinematics and Dynamics
* Robot Control
* Autonomous Mobile Robots
* Advanced Robotics
